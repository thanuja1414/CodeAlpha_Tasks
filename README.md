# Hangman Game

## Project Overview

The Hangman Game is a simple text-based word guessing game developed using Python. The program randomly selects a word from a predefined list, and the player must guess the word one letter at a time before running out of attempts.

## Objective

The objective of this project is to demonstrate the use of Python programming concepts such as loops, conditional statements, lists, strings, and the random module.

## Features

* Random word selection
* Letter-by-letter guessing
* Limited number of attempts
* Win and lose conditions
* User-friendly command-line interface

## Technologies Used

* Python 3 in VSCODE
* Random Module

## How to Run

1. Install Python or download a code editor (say Visual Studio Code) on your system.
2. Save the source code as `hangman.py` or any other file name you like.
3. Open a terminal or command prompt or code editor.
5. Run the program.


## Sample Output

Welcome to Hangman

Word: _ _ _ _ _ _
Attempts Left: 6

Enter a letter: p

Word: p _ _ _ _ _
Attempts Left: 6

## Learning Outcomes

* Understanding loops and conditions
* Working with strings and lists
* Using Python's random module
* Building interactive console application.

# Stock Portfolio Tracker

## Project Overview

The Stock Portfolio Tracker is a Python application that calculates the total value of stocks owned by a user based on predefined stock prices and quantities entered by the user.

## Objective

The objective of this project is to demonstrate the use of dictionaries, loops, arithmetic operations, and file handling in Python.

## Features

* Stores stock prices using a dictionary
* Accepts stock names and quantities from the user
* Calculates total investment value
* Saves results to a text file
* Simple command-line interface

## Technologies Used

* Python 3 in VSCODE

## How to Run

1. Save the source code as `portfolio_tracker.py` or any other preferred file name.
2. Open terminal or command prompt or code editor.
4. Run the program.

# Basic Chatbot

## Project Overview

The Basic Chatbot is a simple rule-based conversational application developed in Python. The chatbot responds to predefined user inputs using conditional statements.

## Objective

The objective of this project is to understand the fundamentals of conversational AI and implement a basic chatbot using Python.

## Features

* Responds to greetings
* Answers predefined questions
* Interactive conversation
* Exit command support
* Simple and easy-to-understand implementation

## Technologies Used

* Python 3 in VSCODE

## How to Run

1. Save the source code as `chatbot.py`or any other preferred file name.
2. Open terminal or command prompt or code editor.
3. Run the program.


## Sample Output

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: what is your name
Bot: I am a Python Chatbot.

You: bye
Bot: Goodbye!

## Learning Outcomes

* Conditional Statements
* User Interaction
* While Loops
* Basic Conversational Logic
